# PriestPower

PriestPower is a priest-focused addon for Turtle WoW (1.12.1), adapted from the popular PallyPower addon. It helps priests manage their buffs (Fortitude, Spirit, Shadow Protection) and specialized champion spells in a raid or party environment.

## Features

- **Buff Management**: Easily assign and track Power Word: Fortitude, Divine Spirit, and Shadow Protection across raid groups.
- **Champion Spells**: Support for Proclaim Champion, Champion's Grace, Empower Champion, and Revive Champion.
- **Raid Coordination**: Syncs with other PriestPower users to avoid overlapping assignments.
- **HUD/Buff Bar**: A compact UI to monitor the status of your assigned buffs and champion timers.

## Commands

- `/prip`: Toggle the main PriestPower configuration frame.
- `/priestpower`: Toggle the main PriestPower configuration frame.
- `/prp`: Toggle the main PriestPower configuration frame.
- `/prip debug`: Toggle debug mode.

## Installation

1. Download the addon.
2. Extract the `PriestPower` folder into your `Interface\AddOns` directory.
3. Ensure the folder name is exactly `PriestPower`.
